This project is a LSTM based custom text generator trained over 20's of Statement of purpose text file (10k words) to generate a sequence of meaningful text 

## Dependences : 
<br>
- Keras (tensorflow-gpu)
- Numpy
<br>

# Cleaning data: 

python clean.py 

# Training 

python Training.py
